Создай PHP-скрипты для парсинга данных с сайта https://fcakron.ru/novosti/ и скачивания медиафайлов. Скрипты должны быть разделены на две части:

#### 1. Скрипт для парсинга данных (parse.php):
- Используй библиотеку Guzzle для HTTP-запросов и Symfony DomCrawler для парсинга HTML.
- Парси все страницы с новостями, начиная с https://fcakron.ru/novosti/ и далее по пагинации (например, https://fcakron.ru/novosti/page/2 и т.д.).
- Для каждой новости извлекай данные с общей страницы:
  - Краткое название (из тега strong class="title"). Сохрани в столбце shortTitle.
  - Краткое описание (из тега div class="description"). Сохрани в столбце previewText.
  - Изображение (из тега img class="cover"). Сохрани в столбце previewImage.
  - Теги (из тега ul class="tag-cloud", содержимое тега a). Если тегов несколько, сохрани их через запятую. Сохрани в столбце tags.
  - Идентификатор матча (из параметра gameId в ссылке тега a class="linked_post"). Сохрани в столбце gameId.
  - Дату публикации (из тега span class="date"). Преобразуй в DateTime и сохрани в столбце datePublication.
- Переходи на внутреннюю страницу новости по ссылке из атрибута data-href в div class="item" и извлекай:
  - Полное название (из тега section class="news_header" div class="h2"). Сохрани в столбце fullTitle.
  - В зависимости от тегов:
    - Если тег "Фото": извлеки все ссылки на изображения из атрибутов data-src в тегах img class="owl-lazy". Сохрани их через запятую в столбце images.
    - Если тег "Видео": извлеки ссылку на видео из тега iframe. Сохрани в столбце link.
    - Для остальных тегов: сохрани HTML-содержимое тега main в столбце text. Если в содержимом есть теги img, извлеки ссылки на изображения и сохрани их через запятую в столбце images, удалив теги img из HTML.
- Сохраняй данные в три CSV-файла в зависимости от тегов:
  - Если тег "Фото" — сохрани в photo.csv.
  - Если тег "Видео" — сохрани в video.csv.
  - В остальных случаях — сохрани в news.csv.

#### 2. Скрипт для скачивания медиафайлов (download.php):
- Читай CSV-файлы (photo.csv, video.csv, news.csv) и скачивай все медиафайлы (изображения и видео), указанные в столбцах previewImage, images и link.
- Сохраняй файлы локально в папку downloaded_media, сохраняя иерархию папок, как в URL. Например:
- Файл https://fcakron.ru/wp-content/uploads/2025/03/1-8-1-800x1200.jpg должен быть сохранен в папку downloaded_media/photos/2025/03/.

#### Дополнительные требования:
- Обеспечь подробные комментарии в коде.
- Добавь обработку ошибок (например, если страница недоступна или данные отсутствуют).
- Убедись, что скрипты могут работать с большим объемом данных (например, с 362 страницами новостей).
- Если возможно, добавь возможность использования прокси для обхода блокировок.

Сгенерируй два отдельных PHP-скрипта: parse.php и download.php.
